🎯 Objetivo SMART
Específico (Specific):
Desarrollar un modelo de machine learning supervisado que prediga con alta precisión la probabilidad de diagnóstico de diabetes en mujeres de ascendencia Pima, utilizando las variables clínicas disponibles en el dataset (glucosa, presión arterial, grosor del pliegue cutáneo, insulina, IMC, función pedigree de diabetes, edad y número de embarazos).

Medible (Measurable):
El desempeño del modelo se evaluará con las siguientes métricas:

Exactitud (Accuracy) ≥ 80%

Área bajo la curva ROC (AUC-ROC) ≥ 0.85

Sensibilidad (Recall) ≥ 0.75 para identificar correctamente casos positivos de diabetes

Alcanzable (Achievable):
Se emplearán técnicas de preprocesamiento (imputación de valores faltantes, escalado de características) y se probarán al menos 3 algoritmos de clasificación (Regresión Logística, Random Forest y XGBoost) con validación cruzada estratificada para garantizar resultados robustos y replicables.

Relevante (Relevant):
Este proyecto contribuye al ámbito de la salud preventiva, proporcionando una herramienta de apoyo para la detección temprana de diabetes en poblaciones con alto riesgo, alineándose con los objetivos de la Organización Mundial de la Salud sobre el control de enfermedades no transmisibles.

Temporal (Time-bound):
El desarrollo completo del proyecto, incluyendo análisis exploratorio, construcción de modelos, evaluación y documentación en este repositorio, se completará en un plazo de 4 semanas a partir de la fecha de inicio.
